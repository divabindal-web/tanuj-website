# Tanuj Shori — Personal Brand Website

Multi-page personal brand site for Tanuj Shori, Founder & CEO of Square Yards.

## Pages
- `index.html` — Home (hero, metrics, thinking, journey, ecosystem, featured articles, press, connect)
- `articles.html` — Articles archive with filter
- `media.html` — Press & media coverage
- `awards.html` — Awards timeline

## Deploy
Static site — no build step. Deploys instantly on Vercel.
All HTML files are at the root; `shared.css` and `shared.js` are shared across pages.
